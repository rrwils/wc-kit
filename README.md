# WC Kit

An interactive project for visualizing football kit data in 3D.

The app uses:

- **React** for the UI
- **Three.js** and **React Three Fiber** for rendering 3D scenes
- **D3** for mapping and visualizing data along three axes

It loads tournament kit data and allows a three-axis mapping system.

## Project structure

- `src/` - main React application code
- `data/` - tournament kit data used by the app
